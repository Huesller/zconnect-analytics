# P1.1 — Inteligência Comercial Essencial

## Objetivo

Adicionar ao Z Connect Analytics uma primeira camada de ranking comercial para apoiar decisões de compra, cadastro e priorização de produtos.

## Funcionalidades implementadas

### 1. Produtos Mais Quentes

Novo painel no dashboard com score comercial:

```txt
score = aberturas + carrinhos × 3 + cotações × 10
```

Campos exibidos:

- produto
- aberturas
- carrinhos
- cotações
- score
- conversão no modal
- último sinal no modal

### 2. Produtos Mais Cotados

Novo painel focado nos produtos com maior quantidade em cotações WhatsApp.

Campos exibidos:

- produto
- cotações
- carrinhos
- aberturas
- score no modal
- conversão no modal

### 3. Demandas Sem Resultado

Novo painel para buscas que não retornaram produtos.

Campos exibidos:

- termo pesquisado
- ocorrências
- empresas distintas
- consultores distintos
- última busca no modal

## UX

Foi criada uma nova seção:

```txt
Inteligência comercial P1.1
```

A seção foi posicionada antes dos rankings antigos de produtos para dar mais destaque às métricas comerciais.

Os cards são clicáveis e abrem modais detalhados com busca interna.

## Observações

Esta entrega não altera o catálogo, Apps Script ou pipeline de coleta.

Ela utiliza somente eventos já existentes:

- `product_open`
- `add_to_cart`
- `whatsapp_quote`
- `search_no_results`

## Critério de aceite

A sprint P1.1 pode ser considerada aceita quando, em produção, o dashboard exibir corretamente:

- Produtos Mais Quentes
- Produtos Mais Cotados
- Demandas Sem Resultado

com dados reais dentro dos filtros atuais de período, empresa e consultor.
