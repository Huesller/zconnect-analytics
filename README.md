# Z Connect Analytics V8

Painel atualizado para acompanhar empresas/clientes sem login.

## Principais telas

- Empresas identificadas
- Jornada do cliente
- Carrinho abandonado
- Ranking empresas
- Ranking consultores
- Produtos enviados
- Buscas mais feitas
- Buscas sem resultado

## Publicação

```bash
npm install
npm run build
git add .
git commit -m "Sprint V8 - Analytics por empresa"
git push origin main
```

## Google Apps Script

Copie o conteúdo de:

```txt
GOOGLE_APPS_SCRIPT_V3_CLIENTES.js
```

cole no Apps Script e publique uma nova implantação.
